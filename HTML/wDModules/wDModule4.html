Module 4: Make it Bling

HTML Refresher

Hyper-text Markup Languages. There are multiple different markup languages, but HTML is the most widely used for website development. Literally marks-up your content, providing an underlying hierachy to the information and assets used. Most important I remember are divs, headers and img tags.

CSS Refresher

Cascading Style Sheets. Linked to HTML to provide visual styling of content. Some of the most important CSS to my knowledge and opinions are flexbox/grids for laying out content, as well the more graphic design elements such as fonts, colours and other visual assets.

WWW HTML Tag

This relay-based interactivity was intended to practise a basic markup structure to seperate content when building websites. My main issue here were the differences between section and divs. Using tabs and indentations really helped to view each start and end tag, just like in real code.

Intial Prototyping

Though this was not a structured crazy-8s prototyping activity, it was still the first introduction to rapid style layout brainstorming. This was completed to use as a resource to then mockup using paper tags, as stated above in the tag activity.
