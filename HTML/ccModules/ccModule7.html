P5 Introduction

P5 is a Javascript, or JS, library that essentially has many prewritten functions that make graphics elements more accessilbe and easy to code. P5 is intended to be integrated with rgular vanilla HTMl, CSS and JS, to create creative coding projects, web sketches and other interactive displays.

Processing

Processing is the software used to code P5 and easily create and view P5 sketches. It is an integrated development environment, or IDE, that is useful for visual contexts of coding, and creative coding projects. We had to add P5 specifically during the setup.

Circles

Learning P5 started by becoming familiar with shapes and how to place objects within the canvas, or sketch. This image shows the result of creating 60 circles per second, at the mouses X and Y coordinates. You can control the colour fill, diameter and location of circles.

Triangles

To create another shape, triangles in P5, you need three points, rather than just the one for circles. Each coordinate is one vertex of the triangle. If you set one of those coordinates to be wherever the mouse is, you get a result as pictured, that follows mouse movement.

Drawing, and a Portrait

P5 can also create lines from the mouse, from the previous mouse point to current point, like a simple paintbrush tool in digital painting software. This allows alot of creative freedom. As a task, I combined a circle with mouse line drawing to create an elementary portrait of a classmate.
